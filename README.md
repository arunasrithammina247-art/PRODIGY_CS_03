PRODIGY_CS_03
Advanced Password Security Analyzer

This project is an advanced Password Strength Checker developed using Python as part of the Prodigy InfoTech Cyber Security Internship.

The tool analyzes password security using multiple cybersecurity techniques such as entropy calculation, crack-time estimation, cryptographic hashing, pattern detection, and secure password generation.

Features
Password Strength Analysis
Entropy Calculation
Crack Time Estimation
Sequential Pattern Detection
Repeated Character Detection
Keyboard Pattern Detection
Secure Password Generator
SHA-256 / SHA-512 / MD5 / SHA1 Hashing
JSON Security Report Generation
Military-Grade Password Classification
